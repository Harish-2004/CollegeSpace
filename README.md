## Getting Started

### Prerequisites

- Node.js (v14 or above)
- npm

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Harish-2004/CollegeSpace.git
   cd CollegeSpace
   ```

2. Install dependencies for the main project:
   ```sh
   npm install
   ```

3. ( backend folder)
   ```sh
   cd backend
   npm install
   cd ..
   ```

4. (Chatrooms folder)
   ```sh
   cd Chatrooms
   npm install
   cd ..
   ```

### Running the App

- To start the App:
  ```sh
  npm start
  ```

- To start the backend server:
  ```sh
  cd backend
  node index.js
  ```

- To start the chat server:
  ```sh
  cd Chatrooms
  node server.js
  ```
